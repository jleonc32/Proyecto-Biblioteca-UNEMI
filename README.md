# 📚 Sistema de Biblioteca - UNEMI

Proyecto de Programación Orientada a Objetos que gestiona libros, usuarios y préstamos.

### 🚀 Retos Implementados:
- **Nivel 1:** Estructura base de clases (Libro, Persona, Estudiante, Profesor).
- **Nivel 2 (RETO):** Catálogo de libros ordenado alfabéticamente.
- **Nivel 3:** Lógica de préstamos y cálculo de multas por retraso.
- **Extra:** Generación de datos aleatorios con la librería `Faker`.
